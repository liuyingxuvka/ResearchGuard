# TraceGuard README Hero Design Note

## Project Summary

TraceGuard is a local Python package and Codex skill for turning noisy multi-source evidence into bounded event traces and temporal storylines.

## Target Users

AI-assisted researchers, investigators, maintainers, and developers who need to preserve evidence boundaries while reconstructing timelines or storylines.

## Core Problem

Scattered sources can be mistaken for final claims. TraceGuard keeps the chain explicit: source, evidence, event, trace, gap, contradiction, and claim boundary.

## Core Workflow

Sources and notes flow through validation gates into an ordered storyline ribbon. Gap and contradiction markers remain visible outside the safe claim boundary.

## Hero Tagline

A local evidence-to-storyline guard for noisy sources, temporal traces, and cautious claims.

## Visual Concept

A bright wide product-object banner showing source cards on the left, verification gates in the center, an ordered temporal ribbon, and a safe claim boundary on the right.

## Image Keywords

source cards, evidence gates, temporal storyline, gap markers, contradiction markers, safe claim boundary, frosted glass, brushed metal, soft blue-green light.
