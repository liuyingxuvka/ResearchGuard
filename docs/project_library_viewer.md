# Project Library Viewer

The Project Library Viewer is LogicGuard's internal read-oriented surface for
project and source inspection. It presents project/source cards, provenance,
model status, important nodes, risk nodes, source dates, coverage periods, and
package operations.

Opening a source renders the single recommended top-level logic graph. The
viewer does not offer competing graph-mode tabs and does not alter the
canonical source model merely to change presentation.

Use the internal `project-library-viewer` route from `$logicguard`; it is not a
separately installed Codex Skill.
