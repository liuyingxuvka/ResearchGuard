# Logic Writing Workflow

Logic Writing is a separate writing orchestrator. ResearchGuard supplies
research-quality checks: SourceGuard plans discovery, LogicGuard licenses the
argument structure, and TraceGuard reconstructs time-ordered evidence and
qualitative causal storylines.

During non-trivial work, use an AI-selected Mermaid diagram or table when it
materially clarifies an argument tree, source path, timeline, gap, document
handoff, or synthesis route. The visual explains the current model; it does
not replace native checks or source evidence.
